# Ядро-Реліз

Модуль повністю автономний: можна скопіювати папку `ядро-реліз` в інше місце і запускати без доступу до основного проєкту.

## Запуск

```powershell
python launcher.py
```

## Що робить

1. **Створити дашборд**
   - Вказуєте папку з `учасники`/`заявки`
   - Вказуєте робочу папку (output root)
   - Опційно додає вкладку статистики
2. **Перемістити статті**
   - Копіює знайдені статті у `run_dir/Статті`
3. **Зібрати чернетку**
   - Збирає `draft_journal.docx` із `run_dir/Статті`
4. **Виявити авторів**
   - Працює по чернетці та ставить стиль `AUTOR`
5. **Виявити назви**
   - Працює по чернетці та ставить стиль `Назва1`

## Шаблон

Шаблон очікується у:
```
ядро-реліз/assets/templates/Jurnal.dotx
```

## Залежності

- `python-docx`
- `openpyxl`
- `pywin32` (для Word COM у збірці чернетки)
