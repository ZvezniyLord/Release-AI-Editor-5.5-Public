# Використання

## GUI
Запуск:
```
python main.py
```
Вкажи:
- `docx` файл
- `manifest.json` (необов'язково)
- `Table.docx` шаблон

## CLI
```
python main.py --cli input.docx --output output.docx --template Table.docx --manifest manifest.json
```

Параметри `--output`, `--template`, `--manifest` необов'язкові.
