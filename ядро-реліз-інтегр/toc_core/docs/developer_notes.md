# Нотатки розробника

## Word COM
- Використовується лише для вставки таблиці і форматування.
- `ScreenUpdating = False` пришвидшує вставку.
- Protected View може блокувати автоматизацію: файл має бути довірений або розблокований.

## Outline parsing
- Читається `document.xml` та `styles.xml`.
- Визначення рівня:
  - за назвою стилю (Heading 1/2/3, Title, Назва 1),
  - або за `outlineLvl` (0..8).

## Типові помилки
- `WordComError` при відкритті файлу: файл у Protected View або вже відкритий.
- Відсутній шаблон `Table.docx`.
- Немає `python-docx` — тоді outline парситься лише через XML (не потрібен Word COM).
