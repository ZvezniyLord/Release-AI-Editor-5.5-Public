# RECENT FIXES (2026-03-23)

## 1) Live View у launcher
- Команди Python з GUI launcher запускаються у небуферизованому режимі.
- Застосовано:
  - ключ `-u` для Python-процесу;
  - змінна середовища `PYTHONUNBUFFERED=1`.
- Ефект: логи й прогрес у вікні launcher оновлюються в реальному часі.
- Логи в launcher завжди ввімкнені.
- Додано перемикач у toolbar: `Видимий Word (COM)` (увімк./вимк.), який керує `WORD_COM_VISIBLE`.

## 2) Вирівнювання front-matter у normalizer (`block_order`)
- Посилено пошук заголовка:
  - пріоритет абзацу зі стилем назви (`Назва1` / `TITLE` / `Heading 3`);
  - точний текстовий збіг;
  - fuzzy fallback (SequenceMatcher) для кейсів з відмінностями після нормалізації.
- Розширено пошук блоку авторів/афіліації перед основним текстом.
- Додано нормалізацію порожніх абзаців у прологу статті:
  - `UDC` -> 1 порожній абзац
  - блок авторів/афіліації -> 1 порожній абзац
  - назва -> 1 порожній абзац

## 3) `shapka coverage`: показ необроблених файлів
- У режимі обробки директорії після рядка coverage тепер виводиться список файлів, де авторів не знайдено:
  - формат: `[shapka] без авторів: file1.docx, file2.docx, ...`

## 5) Технічні логи normalizer
- За замовчуванням `Статті_норм/logs` більше не формується.
- Для діагностики можна явно увімкнути: `--debug-logs`.
- У корені `run_dir` формується один зведений файл: `normalize_debug.json`.

## 6) Правило `*` у ПІБ
- `*` у ПІБ не видаляється під час обробки `shapka`.
- Це семантичний маркер (померла особа), тому зберігається в результаті.

## 7) Prefix перед HTTP у references
- Для рядків references префікс вставляється безпосередньо перед `http://` або `https://`, а не на початок рядка.
- Визначення префікса:
  - якщо URL містить `doi.org` -> `DOI: `
  - інакше -> `URL: `
- Якщо безпосередньо перед посиланням уже є `DOI`/`URL` (з `:` або без), дубль не додається.

## 8) Модульність `normalize_docx` (етап 1)

  - `core/normalize_docx_parts/reference_rules.py`

  - `core/normalize_docx_parts/word_replace.py`
- `core/normalize_docx.py` працює як фасад і використовує ці модулі.
